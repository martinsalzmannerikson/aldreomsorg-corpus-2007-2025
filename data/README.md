# Data layout

The dataset lives in `corpus_2026/`.

Start here:

- `corpus_2026/metadata/curated_sections.csv`
- `corpus_2026/texts/curated_sections/`
- `corpus_2026/texts/curated_chunks_jsonl/curated_chunks.jsonl`

The curated layer is the recommended layer for analysis. Other files in `corpus_2026/metadata/` and `corpus_2026/texts/` are included to support traceability, audit, and alternative workflows.

No original PDFs are stored in this repository.
